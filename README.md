# digital-literacy-project
Digital Literacy Project focuses on developing essential digital skills such as creating online profiles, using coding and collaboration platforms, practicing professional communication, and understanding cyber safety. It helps students become responsible, skilled, and confident users of digital technology.

1. Hands-on Training on Presentation Software
This module focused on using tools like PowerPoint or Prezi to create effective presentations. It covered slide design, visual elements, and clear communication of ideas.

2. Building a Digital Portfolio
This module emphasized creating an online portfolio to showcase skills, projects, and achievements. It helps students build a professional online presence for internships and jobs.

3. Programming Platforms & Collaborative Learning Platforms
Students were introduced to platforms like GitHub, Kaggle, and HackerRank for coding practice, project sharing, and collaborative learning.

4. Email Etiquette & Basics of Social Media Etiquette
This module highlighted professional communication through emails and responsible behavior on social media, including tone, privacy, and online conduct.

5. Awareness on Cyber Crime and Financial Fraudulent
It focused on identifying cyber threats such as phishing, scams, and fraud, along with safety practices like secure passwords and safe online transactions.

✅ Overall: These modules together build essential digital skills, communication abilities, and awareness needed for safe and professional online engagement.
